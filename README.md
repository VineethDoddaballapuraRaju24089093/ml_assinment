ML Tutorial: Gradient Boosting
Focus: How Learning Rate and Number of Trees Affect Performance
Real-world example: Apple stock movement prediction with XGBoost

## Gradient Boosting Tutorial

## Overview
This project demonstrates Gradient Boosting with a focus on how learning rate and number of trees affect performance.

##Contents
- Jupyter Notebook (main tutorial)
- PDF / Web tutorial
- Code for experiments
- Dataset (generated + stock data)

## How to Run

1. Install dependencies:
pip install numpy pandas matplotlib scikit-learn xgboost yfinance

2. Run notebook:
Open mlassignment.ipynb and run all cells.

## Key Concepts
- Gradient Boosting
- Bias-Variance Trade-off
- Hyperparameter tuning
- Learning rate vs number of trees

## Results
- Lower learning rate + more trees improves generalization
- High learning rate causes overfitting
- Stock prediction remains challenging
